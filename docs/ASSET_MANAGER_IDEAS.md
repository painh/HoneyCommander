# Game Asset Database Manager - Feature Ideas

Commander를 게임 어셋 데이터베이스 관리자로 확장하기 위한 기능 제안서

---

## 1. 태그 시스템 (Core)

**설명**: 파일에 커스텀 태그를 추가/제거하고 태그로 검색

- 다중 태그 지원 (`character`, `enemy`, `boss` 등)
- 태그 자동완성
- 태그 색상 지정
- 계층형 태그 (`character/player`, `character/npc`)

---

## 2. 카테고리 분류

**설명**: 어셋 유형별 자동/수동 분류

- 프리셋 카테고리: `Sprite`, `Tilemap`, `UI`, `VFX`, `SFX`, `BGM`, `3D Model`
- 커스텀 카테고리 생성
- 파일 확장자 기반 자동 분류

---

## 3. 메타데이터 자동 추출

**설명**: 파일에서 유용한 정보 자동 추출 및 표시

- 이미지: 해상도, 색상 깊이, 투명도 여부
- 오디오: 길이, 샘플레이트, 비트레이트
- 3D: 폴리곤 수, 텍스처 크기
- 공통: 파일 크기, 생성일, 수정일

---

## 4. 스마트 컬렉션

**설명**: 가상 폴더로 파일을 논리적으로 그룹핑

- 수동 컬렉션: 드래그&드롭으로 추가
- 스마트 컬렉션: 조건 기반 자동 수집 (예: "태그가 'boss'이고 해상도가 512x512 이상")
- 프로젝트별 컬렉션

---

## 5. 별점/평점 시스템

**설명**: 어셋 품질 평가

- 1~5 별점
- 빠른 별점 부여 (키보드 숏컷 1-5)
- 별점 기준 정렬 및 필터

---

## 6. 색상 팔레트 분석

**설명**: 이미지에서 주요 색상 추출

- 상위 5~10개 색상 추출
- 색상으로 어셋 검색 (비슷한 색조 찾기)
- 프로젝트 색상 일관성 확인

---

## 7. 중복 파일 감지

**설명**: 중복/유사 어셋 찾기

- 해시 기반 완전 중복 감지
- 이미지 유사도 기반 감지 (pHash)
- 이름만 다른 동일 파일 찾기

---

## 8. 버전 관리

**설명**: 같은 어셋의 여러 버전 추적

- 버전 히스토리 보기
- 버전 간 비교 (이미지 diff)
- 이전 버전 복원
- 버전 네이밍 규칙 설정

---

## 9. 사용 현황 추적

**설명**: 어셋이 어디에서 사용되는지 추적

- 프로젝트 연결
- 사용 횟수 통계
- 미사용 어셋 찾기
- 의존성 그래프

---

## 10. 고급 검색 및 필터

**설명**: 다양한 조건으로 어셋 검색

- 복합 조건 (AND/OR)
- 저장된 검색 쿼리
- 날짜 범위, 크기 범위
- 정규식 지원

---

## 11. 배치 작업

**설명**: 여러 파일에 일괄 작업

- 배치 태그 추가/제거
- 배치 이름 변경 (패턴 기반)
- 배치 리사이즈/포맷 변환
- 배치 메타데이터 수정

---

## 12. 노트 및 설명

**설명**: 어셋에 메모 추가

- 마크다운 지원 노트
- 사용 가이드라인
- 저작권/라이선스 정보
- 제작자 크레딧

---

## 13. 관련 어셋 연결

**설명**: 연관된 어셋들을 링크

- 수동 연결 (캐릭터 스프라이트 ↔ 포트레잇)
- 자동 추천 (이름 유사도 기반)
- 어셋 패밀리 그룹

---

## 14. 스프라이트 시트 분석

**설명**: 스프라이트 시트 전용 기능

- 그리드 오버레이
- 개별 프레임 미리보기
- 애니메이션 재생 미리보기
- 프레임 정보 (행, 열, 크기)

---

## 15. 통계 대시보드

**설명**: 어셋 라이브러리 현황 파악

- 총 어셋 수, 카테고리별 분포
- 용량 사용량 분석
- 최근 추가/수정 어셋
- 태그 클라우드

---

## 16. 즐겨찾기 및 최근 사용

**설명**: 빠른 접근

- 즐겨찾기 폴더/파일
- 최근 본 어셋 히스토리
- 핀 고정 기능
- 자주 사용하는 어셋 추천

---

## 17. 내보내기 기능

**설명**: 어셋 패키징 및 공유

- 선택한 어셋 + 메타데이터 내보내기
- 프로젝트별 어셋 묶음 내보내기
- 다양한 포맷 지원 (ZIP, 폴더 복사)
- 메타데이터 JSON/CSV 내보내기

---

## 18. AI 자동 태깅 (실험적)

**설명**: 머신러닝 기반 자동 분류

- 이미지 내용 분석 (캐릭터, 배경, 아이템 등)
- 스타일 감지 (픽셀아트, 일러스트 등)
- 태그 제안

---

## 19. 어셋 스토어 연동

**설명**: 외부 어셋 마켓플레이스 정보 관리

- 구매 정보 기록
- 라이선스 유형 추적
- 출처 URL 저장
- 업데이트 알림

---

## 20. 사이드바 속성 패널

**설명**: 선택한 어셋의 상세 정보 표시/편집

- 모든 메타데이터 표시
- 태그 편집
- 노트 편집
- 썸네일 큰 미리보기
- 관련 어셋 목록

---

## 구현 우선순위 제안

### Phase 1 - Core (기반)
1. 태그 시스템
2. 사이드바 속성 패널
3. 노트 및 설명
4. 즐겨찾기

### Phase 2 - Organization (정리)
5. 카테고리 분류
6. 스마트 컬렉션
7. 고급 검색 및 필터
8. 별점 시스템

### Phase 3 - Analysis (분석)
9. 메타데이터 자동 추출
10. 색상 팔레트 분석
11. 중복 파일 감지
12. 통계 대시보드

### Phase 4 - Advanced (고급)
13. 배치 작업
14. 버전 관리
15. 관련 어셋 연결
16. 스프라이트 시트 분석

### Phase 5 - Extended (확장)
17. 사용 현황 추적
18. 내보내기 기능
19. 어셋 스토어 연동
20. AI 자동 태깅

---

## 데이터 저장 방식 고려

| 방식 | 장점 | 단점 |
|------|------|------|
| SQLite DB | 빠른 검색, 복잡한 쿼리 | 파일 이동 시 동기화 필요 |
| Sidecar 파일 (.meta) | 파일과 함께 이동 | 검색 느림 |
| XMP 메타데이터 | 표준, 이미지에 내장 | 지원 포맷 제한 |
| Hybrid | 둘 다 사용 | 복잡도 증가 |

**추천**: SQLite + 선택적 sidecar 내보내기
